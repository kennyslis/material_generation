# MatterGen二维材料生成与迭代训练完整流程

## 概述

完整的MatterGen材料生成、筛选和迭代训练系统，能够生成符合以下三个约束条件的二维材料：
- **元素数量 ≤ 3** (合成性约束)  
- **Energy above hull < 0.1 eV/atom** (稳定性约束)
- **Formation energy < 0.0 eV/atom** (热力学稳定性约束)

## 完整流程概览

```
1. 生成材料 → 2. 筛选材料 → 3. 准备训练数据 → 4. 迭代训练 → 5. 验证优化
```

## 第一步：材料生成

### 1.1 基础材料生成

使用预训练的MatterGen模型生成候选材料：

```bash
# 使用chemical_system_energy_above_hull模型（推荐）
python run_mattergen_generation.py \
    --checkpoint chemical_system_energy_above_hull \
    --num-samples 100 \
    --batch-size 8 \
    --max-elements 3 \
    --energy-above-hull 0.1 \
    --force-2d
```

**输出结果**：
- `mattergen_output/cif_files/` - 生成的CIF结构文件
- `mattergen_output/analysis/` - 初步分析结果
- `mattergen_output/filtered/stable_synthesizable_materials.csv` - 通过前两个约束的材料

### 1.2 可用的预训练模型

| 模型名称 | 支持约束 | 推荐用途 |
|---------|---------|---------|
| `mp_20_base` | 无约束 | 基础生成 |
| `chemical_system_energy_above_hull` | 元素数量 + Energy above hull | **推荐使用** |

## 第四步：MatterGen迭代训练

### 4.1 配置训练参数

创建训练配置文件 `configs/iterative_training.yaml`：

```yaml
# 迭代训练配置
model:
  checkpoint_path: "checkpoints/chemical_system_energy_above_hull"  # 基础模型
  condition_on_property: true
  
data:
  dataset_name: "high_quality_2d_materials"
  train_csv: "training_data/high_quality_2d_materials/train.csv"
  properties_csv: "training_data/high_quality_2d_materials/properties.csv"
  batch_size: 8

trainer:
  max_epochs: 50
  learning_rate: 1e-4
  precision: "16-mixed"
  devices: 1
  
constraints:
  max_elements: 3
  energy_above_hull: 0.1
  formation_energy: 0.0
```

### 4.2 开始迭代训练

```bash
# 第一轮迭代训练
python iterative_training.py \
    --config configs/iterative_training.yaml \
    --iteration 1 \
    --base-checkpoint chemical_system_energy_above_hull

# 自动迭代训练（推荐）
python iterative_training.py \
    --config configs/iterative_training.yaml \
    --auto-iterate \
    --max-iterations 5 \
    --target-success-rate 0.8
```

### 4.3 迭代训练流程

```
迭代1: 基础模型 → 生成材料 → 筛选 → 训练数据 → 微调模型1
迭代2: 微调模型1 → 生成材料 → 筛选 → 更新训练数据 → 微调模型2  
迭代3: 微调模型2 → 生成材料 → 筛选 → 更新训练数据 → 微调模型3
...
```

## 输出结果解读

### 2D评分系统
**2D评分** = 基于晶格参数c/a比值的二维特征评分：
- c/a > 3.0: 高2D特征 (评分0.7-0.9)
- c/a = 2.0-3.0: 中等2D特征 (评分0.5-0.7)  
- c/a < 2.0: 低2D特征 (评分0.1-0.5)

### 筛选成功率
- **>80%**: 模型已很好学习约束条件
- **50-80%**: 模型正在改善，继续迭代
- **<50%**: 需要调整参数或增加训练数据

## 常见问题与解决方案

### Q1: 生成材料质量不高？
**解决方案**：
1. 降低生成temperature (0.5-0.8)
2. 增加guidance_scale (1.5-2.0)
3. 使用更多高质量训练数据

### Q2: 训练收敛太慢?
**解决方案**：
1. 增加学习率 (1e-3)
2. 减小batch_size (4-6)
3. 使用学习率调度器

### Q3: GPU内存不足?
**解决方案**：
1. 减小batch_size (2-4)
2. 使用gradient checkpointing
3. 启用16位混合精度训练

### Q4: 如何选择最佳模型?
**标准**：
1. 验证集上约束满足率 > 70%
2. 生成材料多样性高
3. Formation energy分布合理 (-3 to 0 eV/atom)
